# 🧠 First: what counts as a “research paper” (for your case)

You don’t need to invent a new model.

A paper is “research” if it includes at least one of:

### ✅ Type 1: New perspective / framework

> You reorganize the problem in a better way

### ✅ Type 2: New comparison / synthesis

> You connect things people usually treat separately

### ✅ Type 3: New insight / trade-off analysis

> You identify a conflict or limitation clearly

### ⚠️ Type 4: New algorithm / experiment

> Too heavy for 2 weeks (avoid unless already prepared)
