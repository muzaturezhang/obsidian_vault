## 限制不要回答指挥太多（这种能否改成通用prompt））
另外怎样才能直接让ai就知道我要干嘛（比如整理时间线））

请你先不要给我下一步指令，也不要假设当前状态已经完整。

我会陆续贴 COMMS / directive / handoff / agent log。你的任务是：

1. 先按时间线整理我贴出的信息；
    
2. 明确哪些是“已发生的事实”，哪些是“后续 directive / 计划”，哪些是“尚未验证的假设”；
    
3. 如果上下文不完整，只能说“基于目前已贴信息的暂定理解”，不要贸然让我执行新的指令；
    
4. 在我明确说“请生成下一步 Codex prompt / agent directive”之前，不要生成操作性指令；
    
5. 你可以帮我解释术语、总结当前状态、指出还缺哪些信息；
    
6. 每次回答先给出：
    
    - 已知事实；
        
    - 时间线；
        
    - 当前不确定点；
        
    - 我现在是否足够生成下一步指令。
        
7. 如果你要建议下一步，必须标注为“建议”，不能写成强制执行。
    

现在请先只做信息整理和自然语言解释。
